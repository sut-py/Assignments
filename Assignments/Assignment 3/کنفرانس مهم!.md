# ورودی

ورودی شامل چندین خط است که در خط اول تعداد ایمیل‌ها $(n)$ و در $n$ خط‌های بعدی، ایمیل‌ها هر کدام در یک سطر قرار گرفته‌اند.
$$1 \le n \le 10000$$

# خروجی
خروجی برنامه‌ی شما باید شامل دامنه‌های ایمیل‌ها (هر کدام در یک خط) باشد. به طوری که در صورتی که یک دامنه چند بار موجود بود، تنها یک‌ بار در خروجی چاپ گردد.

# مثال
## ورودی نمونه ۱
```
5
pat_clifton@patpost.com
jess.the.cat@mailmeow.com
sdselby@patpost.com
sarah_gilbert@mailgreendale.com
reverend.timms@churchglenbridge.com
```

## خروجی نمونه ۱
```
churchglenbridge.com
mailgreendale.com
mailmeow.com
patpost.com
```

همان‌طور که مشاهده می‌شود از ۵ ایمیل داده شده در ورودی، دامنه `patpost.com` دو بار تکرار شده بود ولی در خروجی تنها یک‌ بار نشان داده شده است.
